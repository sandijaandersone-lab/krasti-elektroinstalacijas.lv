this error

Šī kļūda "ERR_SSL_VERSION_OR_CIPHER_MISMATCH" nozīmē, ka jūsu serveris (hosting) neatbalsta modernas SSL/TLS protokola versijas vai šifrus, ko pārlūks pieprasa.

Šī problēma parasti rodas, ja:

  1. Jūsu hostingam ir novecojis SSL sertifikāts vai konfigurācija
  2. Serveris neatbalsta TLS 1.2 vai TLS 1.3
  3. Nav pareizi konfigurēts HTTPS

Risinājums:

Pārbaudiet savu hosting pakalpojumu (piemēram, Netlify, Vercel vai citu) un pārliecinieties, ka:

  1. SSL sertifikāts ir aktīvs un derīgs - lielākā daļa mūsdienu hostingu to nodrošina automātiski
  2. Izmantojiet moderno hosting platformu - ieteicams Netlify vai Vercel, kas automātiski konfigurē SSL

Ātrais risinājums ar Netlify:

  1. Pārejiet uz Netlify
  2. Izveidojiet kontu un pievienojiet savu projektu
  3. Deploy no /dist mapes
  4. Netlify automātiski konfigurēs SSL un HTTPS

Vai arī, ja testējat lokāli - lietojiet tieši HTTP (bez SSL):

  • http://localhost:5173 development režīmā
  • Vai pārbaudiet, vai jūsu hosting providers atbalsta modernas SSL versijas
